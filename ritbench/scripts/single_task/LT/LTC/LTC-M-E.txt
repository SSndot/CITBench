Based on given dataset information, construct a "Layout transformation: Concatenating" task, provide task description, Python code for constructing input tables, and Python code for constructing gt tables
# Task explanation
Layout transformation: Concatenating tasks need to merge certain data columns based on given merging rules.
# Task requirements
1. Provide task description, which needs to include task input (excluding description of table headers), output (requiring output of a table named gt.csv), and processing rules.
