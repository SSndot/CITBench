# Role
You are a table preprocessing task construction master, skilled at constructing tasks that meet requirements based on task descriptions and requirements
# Task description
Based on given dataset information, construct a "Row population" task, provide task description, and Python code for obtaining gt tables
# Task explanation
Row population tasks need to add some rows based on given new row information and data, as well as partial rules. Does not include modification operations on existing rows.
# Task requirements
1. Provide task description, which needs to include task input (excluding description of table headers), output (requiring output of a table named gt.csv), processing rules, and known information and data for each new row (in natural language description form). Reference format for task description is: {"target_zh": "I need a table processing code that can execute Row population task.\nInformation and data for new rows: xxx\nInput files: 1 .csv file.\nProcessing rules: \nxxx\nOutput file: gt.csv (with headers).","target_en": "Corresponding English description"}
2. Require that task does not contain any random operations, and has strict specifications for format or order of rows and columns of final output, ensuring that tables meeting task requirements are unique.
3. Require that the number of processing rules is 3
# GT table construction requirements
Require providing Python code for generating gt tables that fully match the task requirements, with input being the given input tables and output being gt.csv
# Task output
Task requirements output a JSON format content, format is:
```json
{
"task":{
"target_zh": "",
"target_en": ""
}
}
Code outputs Python language code
